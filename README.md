# PS4HEN (Local Fork)

Local copy of [PS4HEN](https://github.com/Scene-Collective/ps4-hen-vtx) — Homebrew ENabler for PlayStation 4 (firmware 5.05 - 12.02).

## Features

- Homebrew Enabler (HEN)
- Plugin system (bundled from ps4-hen-plugins)
- Jailbreak + sandbox escape
- Debug settings access
- External HDD support + format
- VR support
- Remote Package Install
- Rest mode support
- Debug trophies
- UART enabler
- Screenshot bypass
- Remote Play enabler
- ASLR disable

## Supported Firmware

5.05 - 12.02

## Building

Requires [ps4-payload-sdk](https://github.com/Scene-Collective/ps4-payload-sdk).

```bash
# Install SDK first, then:
make
```
